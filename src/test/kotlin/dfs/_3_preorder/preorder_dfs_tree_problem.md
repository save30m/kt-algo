# DFS 트리 전위 순회 문제

## 문제 설명
아래와 같은 이진 트리가 주어진다:

```
        10
       /  \
      5    20
     / \   /
    3   7 15
```

이 트리를 **깊이 우선 탐색(DFS)** 의 **전위 순회(Preorder Traversal)** 방식으로 순회했을 때 방문 순서를 구하라.

전위 순회 규칙:
1. 현재 노드를 방문한다.
2. 왼쪽 서브트리를 전위 순회한다.
3. 오른쪽 서브트리를 전위 순회한다.
